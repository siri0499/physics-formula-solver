# Physics Formula Solver ⚛️

A simple Python program to calculate basic physics formulas.

## Formulas Included
- Speed = Distance / Time
- Force = Mass × Acceleration
- Kinetic Energy = ½mv²

## How to Run
1. Install Python
2. Run `physics_solver.py`

- CET
- JEE
- Class 11 Physics
